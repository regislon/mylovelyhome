# Comment utiliser cette interface

## Navigation entre les étages

Cliquez sur les boutons **"Entrée"**, **"1er étage"** ou **"Attique"** dans le menu du haut pour naviguer entre les différents niveaux de l'appartement. Le plan d'étage correspondant s'affichera automatiquement dans le panneau de gauche.

## Description générale

Cliquez sur le bouton **"Description générale"** dans le menu du haut pour afficher une vue d'ensemble de l'appartement et sa localisation.

## Exploration des pièces

### Afficher la description d'une pièce
Sur le plan d'étage (panneau de gauche), cliquez directement sur une zone du plan pour afficher la description détaillée de la pièce dans le panneau de droite.

### Vues panoramiques à 360°
- Cliquez sur les **icônes de bonhomme** sur le plan d'étage pour afficher une vue à 360° de l'emplacement sélectionné
![alt text](image-3.png)

- Utilisez votre souris ou votre doigt (sur écran tactile) pour explorer la vue à 360° :
  - **Glissez** pour regarder autour de vous
  - **Molette de la souris** ou **pincement** pour zoomer/dézoomer
  - Cliquez sur les **flèches directionnelles** (Nord ↑, Est →, Sud ↓, Ouest ←) pour vous déplacer vers les pièces connectées
