# Claude Code Configuration

This directory contains configuration files for Claude Code.

## Structure

- `commands/` - Custom slash commands for this project
- `prompts/` - Custom prompts and instructions
- `settings.json` - Project-specific settings

## Usage

### Custom Commands
Create `.md` files in the `commands/` directory to define custom slash commands.
Example: `commands/review.md` becomes `/review`

### Custom Prompts
Store reusable prompts in the `prompts/` directory for common tasks.

### Settings
Configure project-specific behavior in `settings.json`.
